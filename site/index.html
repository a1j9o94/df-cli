Key Features

Parallel Builders

Multiple AI agents build modules simultaneously in isolated git worktrees. Each builder works independently without interfering with others.

Holdout Scenarios

Specs include test scenarios that builders never see. The evaluator validates the built code against these hidden requirements.

Contract-Driven Decomposition

An architect agent breaks specs into modules with precise interface contracts. Builders implement against contracts, not each other.

Budget Control

Set cost limits per build. The pipeline tracks token usage and elapsed time, auto-pausing when budgets are exceeded.
